Project 1: Gitlet (Java)

A version control system similar to Git, I built this program from scratch in Java, when I took CS61BL over Summer 2019.

The program comes with most of Git's functionality, such as branching, merging, and checking-out. The basic structure is that of a 'LinkedList' whereby in the same branch, the current and previous commit's commit ID is noted and is then access when a file is checked out.

Like Git, there is also a 'head' pointer that points to the 'head' commit, which is usually the most recent commit.

Project 3: Refugee Responsibility and Sustainability Index (Python) - In Progress
